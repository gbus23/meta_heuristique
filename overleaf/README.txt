IMAGES POUR OVERLEAF
====================

Ce dossier contient toutes les images nécessaires pour compiler le rapport LaTeX sur Overleaf.

Fichiers inclus:
- comparison_sensors.png : Graphique comparatif VNS vs Recuit Simulé
- comparison_radius.png : Impact des rayons sur le nombre de capteurs
- captANOR150_R2-3.png : Solution pour instance aléatoire (150 cibles, 6 capteurs)
- captANOR400_R1-1.png : Solution captANOR400 avec R=(1,1) : 33 capteurs
- captANOR400_R1-2.png : Solution captANOR400 avec R=(1,2) : 22 capteurs
- captANOR400_R2-2.png : Solution captANOR400 avec R=(2,2) : 9 capteurs
- captANOR400_R2-3.png : Solution captANOR400 avec R=(2,3) : 7 capteurs
- captANOR1600_R2-3.png : Solution pour instance aléatoire (1600 cibles, 33 capteurs)
- grille1010_R2-3.png : Solution pour grille 10x10 (14 capteurs)

Instructions:
1. Uploader rapport.tex sur Overleaf
2. Uploader toutes les images de ce dossier dans le même dossier que rapport.tex
3. Compiler le document

Toutes les solutions ont été générées avec le recuit simulé avec un temps limite de 120 secondes.
